# go_aprender 
Bem-vindo ao repositório oficial do curso básico de Go! Este espaço é dedicado a armazenar todos os códigos e materiais desenvolvidos durante o curso.

O nome *go_aprender* reflete a missão: um comando prático para unir a linguagem *Go* ao processo contínuo de aprendizado.
E também fica um trocadilho entre português e inglês "vá aprender".

## 📖 Sobre o Curso
Este curso é destinado a iniciantes e entusiastas que desejam aprender a linguagem Go de forma sólida.
Utilizei como base os conceitos fundamentais da linguagem, adaptados para uma estrutura de pastas organizada e progressiva.

_*O que você encontra aqui*_:
Exemplos de aula: Códigos escritos durante os vídeos.

Exercícios: Desafios práticos para reforçar a lógica.

Links Úteis: Referências para o Go Playground e documentação oficial.

## 📁 Estrutura de Pastas
Organizamos o código de forma incremental:

01-introducao/: Primeiros passos.

02-fundamentos/: Variáveis, tipos e lógica inicial.
